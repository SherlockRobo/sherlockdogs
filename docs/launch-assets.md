# Launch Assets

Visual assets used by the public beta README and release pages.

| File | Use |
|---|---|
| `docs/assets/sherlockdogs-hero.png` | README hero image |
| `docs/assets/quickstart-flow.png` | Quickstart / release visual |
| `docs/assets/social-preview.png` | Social preview / announcement background |

## Copy Rules

- Use `sherlockdogs-hero.png` at the top of README-style pages.
- Use `quickstart-flow.png` near install instructions.
- Use `social-preview.png` for GitHub social preview, launch posts, and link cards.
- Do not use the old SaaS-card XHS images as the main visual.
- Do not describe Windows as fully equivalent to macOS until real Windows self-chat smoke passes.

## Public Beta Copy

```text
Sherlockdogs turns phone WeChat self-chat items into local Markdown files and optional Codex tasks.

Mac is verified. Windows is packaged for the same path and waiting for real-machine smoke. iOS Shortcut / Inbox remains the fallback.
```
