# Public Beta Launch Checklist

## GitHub

- [ ] README hero renders.
- [ ] README links to macOS and Windows beta folders.
- [ ] Release notes mention Mac verified and Windows smoke pending.
- [ ] `START_HERE.md` exists in both platform folders.
- [ ] `INSTALL_GUIDE_FOR_USERS.png` exists in both platform folders.
- [ ] Windows evidence path says `Export Windows Evidence.cmd`.

## Public Copy

- [ ] Main path is `phone WeChat -> desktop WeChat -> local DB -> Markdown/Codex`.
- [ ] iOS Shortcut / Inbox is described as fallback.
- [ ] Windows is not described as fully Mac-equivalent.
- [ ] No hosted bot or cloud relay is implied.

## Package Policy

- [ ] Do not publish zip/dmg/tar unless explicitly requested.
- [ ] Share the whole platform folder.
- [ ] Keep private paths, account names, credentials, and app DB contents out of public docs.
